# 🚀 Wipro_NGA_Training

The `Wipro_NGA_Training` repository serves as a comprehensive collection of exercises, code samples, and projects developed during the Wipro NGA (NextGen Academy) program. This repository is designed to showcase foundational to intermediate .NET development skills, primarily using C#. It acts as a practical learning resource and a portfolio of completed training modules, demonstrating practical application of various concepts.

## ✨ Features

*   💡 Demonstrates core C# language features and object-oriented programming (OOP) principles.
*   🏗️ Includes various project types, ranging from simple console applications to potentially more complex service examples.
*   ✨ Showcases practical application of common .NET libraries and frameworks.
*   📚 Structured into modules or topics for clear understanding and learning progression.
*   ✅ Follows industry standard coding conventions and best practices for maintainable code.

## 🛠️ Tech Stack

*   **Language:** C#
*   **Framework:** .NET 7.0 SDK
*   **Development Environment:** Visual Studio / Visual Studio Code
*   **Version Control:** Git

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Before you begin, ensure you have the following installed:

*   [.NET 7.0 SDK](https://dotnet.microsoft.com/download/dotnet/7.0)
*   A code editor like [Visual Studio Code](https://code.visualstudio.com/) or [Visual Studio](https://visualstudio.microsoft.com/) (Community Edition is free and recommended for full .NET development).
*   Git (for cloning the repository).

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/YourUsername/Wipro_NGA_Training.git
    cd Wipro_NGA_Training
    ```
    *(Note: Replace `YourUsername` with the actual GitHub username or organization if this repository is hosted under one.)*

2.  **Restore project dependencies:**
    Navigate to the root of the cloned repository and run:
    ```bash
    dotnet restore
    ```

3.  **Build the project:**
    Ensure all projects compile correctly:
    ```bash
    dotnet build
    ```

## 🎮 Usage

Each project or exercise within this repository is typically located in its own subfolder (e.g., `src/HelloWorldApp`, `src/DataStructures`). To run a specific project, navigate to its directory and use the `dotnet run` command.

For example, to run a console application located in `src/HelloWorldApp`:

```bash
# Navigate to the specific project directory
cd src/HelloWorldApp

# Run the application
dotnet run
```

If the project is an ASP.NET Core web application, running `dotnet run` from its directory will typically start a local web server, and you can access it via your web browser (usually at `https://localhost:7001` or `http://localhost:5000`).
